# Project Notes

Personal repository for tracking projects, ideas, and technical notes.

## Structure

```
notes/
├── projects/       # Active project notes
├── templates/      # Note templates
├── archive/        # Completed or inactive projects
└── README.md
```

## Usage

- Create a new project note: Copy `templates/project.md` to `projects/`
- Archive completed projects: Move from `projects/` to `archive/`

## Quick Commands

```bash
# Create a new project note
cp templates/project.md projects/my-new-project.md

# List all projects
ls projects/
```
